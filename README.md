**Jun 2025 - Nov 2025**

A wireless power transfer system for electric vehicles using electromagnetic induction, enabling dynamic charging while in motion.

- 🔋 Designed a resonant inductive coupling circuit for contactless energy transfer
- 🚗 Aimed at eliminating the need to stop for charging — EV charges while driving
- ♻️ Focused on improving energy efficiency and supporting sustainable transportation

### 🌱 Soil Moisture Monitoring System with Temperature and Humidity
**Oct 2024 - Apr 2025**

An automated smart agriculture system that monitors soil moisture, temperature, and humidity in real time to enable efficient irrigation.

- 💧 Integrated soil moisture sensor with DHT11 for combined environmental monitoring
- 📡 Real-time data acquisition for smart irrigation decision-making
- 🌾 Designed for agricultural applications to improve crop productivity and reduce water waste
- 🔔 Alert system triggers when soil conditions go outside the optimal range
